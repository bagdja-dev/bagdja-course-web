# bagdja-course (FE)

Next.js (Pages Router) + Tailwind + TypeScript boilerplate for a course portal (online/offline) + eBook store.

## Run

```bash
cd bagdja-course
npm install
npm run dev
```

## Pages

- `/` Home (hot courses, books, events, testimonials, locations)
- `/books` eBook listing + details
- `/courses` course listing (online/offline) + details
- `/booking/[slug]` booking form (course)
- `/checkout` order summary (placeholder for payment integration)
